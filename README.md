# SecondRecap

| 커밋 유형 | 의미 |
| --- | --- |
| Feat | 새로운 기능 추가 |
| ADD | Feat 이외의 부수적인 코드 추가, 라이브러리 추가, 새로운 파일 생성 시 |
| Fix | 버그, 오류 해결 |
| DEL | 파일을 삭제하는 작업만 수행한 경우 |
| MOD | UI 수정 |
| Comment | 필요한 주석 추가 및 변경 |
| Rename | 파일 또는 폴더 명을 수정하거나 옮기는 작업만인 경우 |
| MOVE | 프로젝트 내 파일이나 코드의 이동 |
| DOCS  | README나 WIKI 등의 문서 개정 |
